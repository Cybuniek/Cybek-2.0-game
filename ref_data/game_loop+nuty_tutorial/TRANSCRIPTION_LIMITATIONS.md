# Ograniczenia ekstrakcji

- Paczka bazuje głównie na analizie wizualnej klatek i widocznych napisów UI.
- Nie wykonano pełnej transkrypcji ścieżki audio.
- Oryginalne filmy nie są dołączone do paczki, żeby utrzymać mały rozmiar.
- Wybrane kadry mają nałożony timestamp w lewym górnym rogu.
- Contact sheety pokazują szerszy przebieg, ale nie zastępują pełnego filmu, gdy potrzebna jest analiza animacji frame-by-frame.
