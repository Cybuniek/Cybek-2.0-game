# Codex Video Reference Pack

Lekki zestaw referencyjny wyciągnięty z filmów:

- `game-loop.mp4` — pełny przebieg pętli: pulpit → wybór utworu → rytm → publikacja/wybór treści → wynik → reakcja świata → powrót do huba.
- `nuty-tutorial.mp4` — samouczek typów nut: pojedyncze, długie, pędzące.

Paczka nie zawiera oryginalnych filmów, żeby pozostała lekka. Zawiera tylko wybrane kadry, contact sheety, indeks czasowy i streszczenie systemów.

## Jak używać w Codexie

1. Najpierw przeczytaj `CODEX_BRIEF.md`.
2. Potem sprawdź `GAMEPLAY_CONTEXT.md`, jeśli potrzebujesz pełniejszego opisu pętli.
3. Do szybkiego przeglądu wizualnego użyj:
   - `assets/contact_sheets/game-loop_contact_sheet.jpg`
   - `assets/contact_sheets/nuty-tutorial_contact_sheet.jpg`
4. Do konkretnych ekranów użyj folderu `assets/frames_selected/`.
5. Strukturalne dane masz w `data/video_context_index.json`.

## Ważne ograniczenie

To jest materiał referencyjny do analizy struktury UX/gameplayu, nie instrukcja klonowania cudzego IP. Przy implementacji używaj własnych nazw, tekstów, grafiki i logiki projektu docelowego.
